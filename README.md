# API Express com MongoDB

API RESTful para gerenciamento de livros com Node.js, Express e MongoDB.

## Tecnologias
- Node.js
- Express
- MongoDB
- Mongoose

## Como usar
1. Clone o repositório
2. 
pm install
3. Configure o arquivo .env
4. 
pm run dev

## Endpoints
- GET /livros
- GET /livros/:id
- POST /livros
- PUT /livros/:id
- DELETE /livros/:id
